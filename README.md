### Install

    $ yarn install

### Configure app

    1. install mysql
    2. you can also download mysql GUI `mysql workbench`.
    3. paste host, user, password, database to `database/mysqlDB.js`
    4. run product.sql

    => test on: http://localhost:3000/api/products

### Start app

    $ npm start

### Start app using Nodemon

    $ npm run dev
